## HiveChat
HiveChat is a group chat mobile application that allows users to communicate with each other in real-time. The app enables users to create or join chat groups and uses AES encryption and decryption algorithm to secure the messages. The appâ€™s user interface is designed to be intuitive and user-friendly.
